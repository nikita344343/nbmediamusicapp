# Пароль приложений (Application Password)

Ты создал пароль приложений для сайта appnbmedia.ru:

- **Имя:** NBM  
- **Пароль:** храни только у себя (в менеджере паролей или записанный в безопасном месте).

## Важно

- **Никогда не добавляй пароль в код** и не коммить его в репозиторий.
- В приложении пароль будет вводить только пользователь (в экране входа), когда мы добавим авторизацию. Хранить его можно в Android Keystore / EncryptedSharedPreferences.
- Сейчас приложение только **читает посты** без авторизации (публичный доступ). Пароль понадобится, когда будем делать: загрузку треков, личный каталог по пользователю, статистику.

## Проверка API

REST API у тебя работает. Ответ от `https://appnbmedia.ru/wp-json/wp/v2/posts` приходит в стандартном формате WordPress (id, date, title, featured_media, author, categories и т.д.). В приложении по умолчанию подставлен URL `https://appnbmedia.ru` — можно сразу открыть приложение и нажать «Загрузить посты».
